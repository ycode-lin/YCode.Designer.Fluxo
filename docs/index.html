YCode.Designer.Fluxo is an exploratory WPF control library for node dragging, viewport orchestration, line rendering, and MVVM-oriented flow editing. It is built as a practical study in graphical software composition rather than a marketing shell.

Capabilities

A WPF canvas study shifting toward a control-first model.

The current branch focuses on moving away from an ad-hoc canvas approach and toward a cleaner control architecture. The result is a repository that is useful both as code and as reference material for developers building their own visual editors.

Viewport control

Built-in transforms manage zoom and pan while the control tracks extent and viewport location as first-class state.

Interactive nodes

Drag events and selection infrastructure make the designer surface behave like an editor, not just a passive visual.

Line strategy

Multiple line types allow the project to explore different connection semantics for flow, graph, and node editor scenarios.

Technical Notes

The repository is small, but the interaction surface is not.

Primary control FluxoDesigner coordinates viewport state, item hosting, drag events, and tool placement.
Layout engine FluxoPanel arranges items and computes extent bounds from rendered children.
Extension helpers Utility methods cover hit testing, safe mouse capture, collection helpers, and visual-tree traversal.
Target framework net8.0-windows with WPF enabled.
